MEUS PROJETOS

PROJETO ANGULAR PETSHOP!

Nesse projeto pude desenvolver uma página de PetShop que fornece informações essenciais sobre cuidados com animais de estimação, incluindo uma API de localização para encontrar PetShops próximos e dados de veterinários. Utilizei o Git para controle de versão, garantindo um desenvolvimento organizado e colaborativo, pois foi feito durante meu curso com mais 2 colegas de sala. Este projeto destaca minha capacidade de integrar tecnologias diversas para criar soluções úteis para amantes de animais de estimação.

git@github.com:Peixoto16/PetsGoo.git

PROJETO PAGINA DE CADASTRO COM SPRING SECURITY E JWT!!

Construí um sistema de login/cadastro do zero! No front, usei HTML, CSS e JS puro – foi desafiador dominar a manipulação DOM sem frameworks. No back, Java com Spring Boot me ensinou muito: Spring Security para proteção, JWT para tokens (aprendi na prática como armazená-los com segurança) e @ControllerAdvice para tratar erros de forma elegante. Organizei tudo no Git com branches separadas e PRs – percebi como isso acelera o trabalho em equipe. Cada bug resolvido me mostrou a importância de um código limpo e documentado!

git@github.com:Peixoto16/LoginAutentication.git
